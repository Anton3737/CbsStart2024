Java світлом код горить,
Класів сила все творить.
Методи, що викликаєм,
Мову швидко опануваєм.

Об'єкти, змінні — це про нас,
Поліморфізм палає враз.
Інтерфейси та потоки,
Вивчимо усі уроки.

Середовище IDE —
Відкриває шлях новий.
В IntelliJ чи Eclipse,
Пишем код, як в чистий сніг.

Абстракція нас чарує,
Наслідування танцює.
Java світ чудовий, брате,
З нею ми не будем спати!
